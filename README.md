Esto es el backend de la pagina web huerto hogar.

Para ejecutar la pagina, hay que seguir los siguientes pasos:

1. Abrir el cmd
2. dirigirte al directorio donde se guardo el repositorio
3. Ejecutar el comando "npm install axios"
4. Ejecutar en otro cmd el comando "curl -X POST http://localhost:3000/api/products/init"

Esto enciende el backend e inicializa la base de datos para almacenar la informacion de los productos.

Mas adelante, para utilizar el frontend en la siguiente pagina, hay que seguir estos pasos (realizar despues de seguir los pasos anteriores):

1. abrir el cmd
2. dirigirte al directorio del repositorio del frontend
3. ejecutar el comando "npm install"
4. ejecutar el comando "npm run dev"

Con esto, finalmente podran probar la pagina web.
